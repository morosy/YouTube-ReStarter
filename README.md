# YouTube-Reset

YouTube の動画を **必ず 0:00 から再生**するための Chrome 拡張機能です．
YouTube の SPA（Single Page Application）構造に対応し，動画遷移時に自動で再生位置をリセットします．

ver 1.3 系より，設定画面による詳細カスタマイズに対応しています．
ver 1.3.5 より，**再生時間の復元**に対応しています．

---

## 機能概要

- YouTube の動画を **必ず 0:00 から再生**
- SPA 遷移対応（関連動画クリック，戻る/進む）
- 完了時にトースト通知（設定に応じて表示）
- 拡張機能アイコンから **ON / OFF 切替**
- OFF → ON 操作時の誤動作防止（再生中の動画には適用しない）
- 設定画面によるトーストの詳細カスタマイズ（位置，大きさ，表示時間，色）
- 保存ボタン方式（変更のみでは保存されない）
- 未保存状態インジケータ（●）
- **再生時間の復元（ver 1.3.5）**
  - 0:00 へリセットする直前の再生時間を記憶
  - ポップアップの「再生時間を復元」ボタンで復元
  - 説明文に復元予定時刻を動的表示（例：`再生時間を 03:21 へ復元します`）
  - 有効化 OFF の場合は復元 UI を非表示

---

## 対応ページ

- `https://www.youtube.com/watch?v=[ID]`

※ 現在は `/watch` ページのみ対象です．
（Shorts や埋め込み動画は未対応）

## インストール方法（開発用）

1. このリポジトリをダウンロード，またはクローン
2. Chrome で以下を開く

`chrome://extensions/`

3. 右上の **「デベロッパーモード」** を ON
4. **「パッケージ化されていない拡張機能を読み込む」**
5. `YouTube-Reset` フォルダを選択

---

## 使い方

### 基本動作

- 有効化 ON の状態で YouTube の動画ページを開く
- 動画が自動的に 0:00 に戻る
- 処理完了後，トースト通知を表示（設定に従う）

### ON / OFF 切替

- Chrome 右上の拡張機能アイコンをクリック
- トグルスイッチで有効化を切り替え

#### OFF 時の挙動

- 動画は一切リセットされない
- トーストも表示されない
- ポップアップ内の「再生時間を復元」UI は非表示

#### OFF → ON 時の挙動

- 切り替えた瞬間に再生中の動画には適用されない
- 次の動画遷移から有効

### 再生時間の復元（ver 1.3.5）

- 拡張機能が 0:00 にリセットする直前の再生時間を記憶します．
- ポップアップの「再生時間を復元」ボタンを押すと，直前に記憶した時刻へ復元します．
- 説明文に復元予定時刻を表示します．復元対象が無い場合は「復元対象なし」と表示します．

---

## 設定画面（ver 1.3 以降）

ポップアップの設定ボタンから設定タブを開くことができます．

### 基本設定

- 有効化（ON / OFF）
- ポップアップ表示 ON / OFF

### 詳細設定（ポップアップ ON のとき表示）

- ポップアップ位置（左上，中央，右上）
- ポップアップの大きさ（スライダー）
- 表示時間（1～10秒）
- 背景色（テンプレカラー＋カラーピッカー）
- フォント色（テンプレカラー＋カラーピッカー）
- プレビューボタン

### 保存仕様

- 「設定を保存」ボタン押下時にのみ保存
- 未保存状態は ● で表示
- 保存完了時は中央上部に `設定を保存しました` と表示

## 技術仕様

- Chrome Extensions Manifest V3
- Content Script による制御
- YouTube SPA 対応
    - History API（`pushState`，`replaceState`）
    - `yt-navigate-finish` イベント
    - `MutationObserver`
- 設定保存
    - `chrome.storage.local`
- ポップアップからの操作
    - `activeTab` を使用して，アクティブタブの content script にメッセージ送信
- 再生制御
    - `HTMLVideoElement.currentTime = 0`

## 注意事項

- 拡張機能を更新した後は，YouTube タブをリロードすると安定します．
- YouTube 側の仕様変更により動作しなくなる可能性があります．
